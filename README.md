# Cyber Security Project

## Project Title

Zero Trust Network Access (ZTNA) with Zscaler: Securing User-to-Application Connectivity.


## Project Overview

This project focuses on implementing Zero Trust Network Access (ZTNA) using the Zscaler security platform to secure user-to-application connectivity. The project explains how organizations can protect applications by verifying users and devices before granting access.

It also demonstrates web application vulnerability analysis, vulnerability scanning using Nessus, and integration of Zscaler with Identity Providers (IdPs) such as Azure AD, Okta, and Active Directory to enable secure authentication and authorization.

## Project Stages

1. UWeb Application Vulnerability Analysis
Identifying common vulnerabilities in web applications such as SQL Injection, Cross-Site Scripting (XSS), and insecure configurations.

2. Vulnerability Scanning using Nessus
Performing automated vulnerability scans to detect security weaknesses in target web applications.

3. ZTNA Implementation with Zscaler
Integrating Zscaler with Identity Providers (IdPs) like Azure AD, Okta, and Active Directory to enforce Zero Trust security policies and provide secure user-to-application access.

## Tools Used

-Browser Developer Tools – To inspect and analyze web application behavior

-Nessus Vulnerability Scanner – To identify vulnerabilities in web applications

-Zscaler Security Platform – To implement Zero Trust Network Access (ZTNA)

-OWASP Security Concepts – To understand common web application security risks
